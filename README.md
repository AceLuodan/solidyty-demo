# solidyty-demo
智能合约入门

智能合约语言solidity入门基础知识

    本项目编写环境为
    Truffle v5.1.45 (core: 5.1.45)
    Solidity - 0.4.22 (solc-js)
    Node v10.15.3
    Web3.js v1.2.


    智能合约交流qq群：786937587

    简单的hello world 入门，学习solidity开发环境的搭建，以及测试

       学习代币合约,开发简单的代币，
       具备初始化，查询余额，转账功能

#### demo03_bloggerCoin
       学习erc20的代币合约
